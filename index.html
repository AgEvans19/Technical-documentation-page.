HTML Introduction

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures content using tags, defining elements like headings, paragraphs, images, and links.

CSS Introduction

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows for the separation of presentation and content, enabling custom designs, responsive layouts, and more.

HTML Elements

Basic HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor/Link
  • <img>: Image
  • <div>: Division

CSS Properties

Basic CSS Properties

  • color: Sets the text color.
  • font-size: Sets the size of the font.
  • margin: Adds space around elements.
  • padding: Adds space inside elements.
  • background-color: Sets the background color.

Examples

HTML Example

                
<!DOCTYPE html>
<html>
    <body>
        <h1>Hello World</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

CSS Example

                
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}

h1 {
    color: #333;
    font-size: 2em;
}

p {
    color: #666;
    font-size: 1em;
}
